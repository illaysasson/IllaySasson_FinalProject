World Happiness Report 2016


The World Happiness Report is a dataset that includes information about a country's happiness in a certain year (In this case 2016). This dataset is the main way to see how does the internet affect the world happiness. The happiness score is built from seven variables: Economy, Social Support, Health, Freedom, Trust, Generosity and Dystopia Residual. In the year 2016, The happiest country in the world is Denmark, and the least happy one is Burundi. Fun fact: Israel is ranked in the 11th spot, which is above countries like the UK, the United States, and a lot more. I took the dataset from the website kaggle, but the data is originally from The World Happiness Report website, where you can find a lot more information about how the data was collected and different Happiness Reports from previous years.
